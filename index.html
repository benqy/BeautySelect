Beautyselect

一个简易的下拉框美化控件，只有几十行，样式可在css中自定义。

Download .zip Download .tar.gz View on GitHub

创建下拉框


<select name="select" id="s1">
  <option value="1">选项1</option>
  <option value="1">选项2</option>
  <option value="1">选项3</option>
  <option value="1">选项4</option>
</select>


$(function() {
  $('#s1').beautySelect();
});
